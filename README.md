# Aware

Agent discovery and registration.

## Backend

### Setup

1. Navigate to the backend directory:

   ```sh
   cd backend
   ```

1. Install dependencies:

   ```sh
   pnpm install
   ```

1. Start the server (development):

   ```sh
   pnpm dev
   ```

   The server will run on `http://localhost:3000` by default.
